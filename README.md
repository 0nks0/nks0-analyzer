# NKS0 — PCAP Analyzer

A web-based network packet capture analyzer built from a defender's perspective.

Drop a `.pcap` or `.pcapng` file and get an instant security analysis — alerts, protocol breakdown, top hosts, and more.

## Features

- 30+ detection rules covering port scanning, brute force, data exfiltration, DNS tunneling, credential exposure, and more
- Severity-ranked alerts (High / Medium / Low / Info) with evidence details
- Protocol distribution and traffic charts
- Top host breakdown by traffic volume
- Export results as JSON

## Status

Active development — more detection rules and visualizations coming.
